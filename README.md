## 特性
- [开源](https://github.com/way-zer/myapp)
- 没广告，免安装
- 使用PWA技术,可**离线运行**
- 支持PC，安卓，苹果多端(添加到桌面堪比原生App)
- 拥有自动导入功能(**应用内无需输入账号密码，安全可靠**)
- 拥有备份同步功能, 课程表不丢失，多端可同步
## 自动导入功能教程
手动登入教务系统，复制课程部分(如同), 拖动或粘贴到小程序特定区域
> 目前仅支持BUPT, 其他学校请提交 Issue 请求适配
